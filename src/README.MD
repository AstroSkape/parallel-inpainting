## Setup

1. Install OpenCV:
   - **macOS:** `brew install opencv`
   - **Linux:** `sudo apt install libopencv-dev`

2. Build:
   ```bash
   mkdir build
   cd build && cmake .. && make
   ```

3. Run:
   ```bash
   ./inpaint
   ```
